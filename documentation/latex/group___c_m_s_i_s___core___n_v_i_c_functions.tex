Set Priority Grouping. 

Sets the priority grouping field using the required unlock sequence. The parameter Priority\-Group is assigned to the field S\-C\-B-\/$>$A\-I\-R\-C\-R \mbox{[}10\-:8\mbox{]} P\-R\-I\-G\-R\-O\-U\-P field. Only values from 0..7 are used. In case of a conflict between priority grouping and available priority bits (\-\_\-\-\_\-\-N\-V\-I\-C\-\_\-\-P\-R\-I\-O\-\_\-\-B\-I\-T\-S), the smallest possible priority group is set.
