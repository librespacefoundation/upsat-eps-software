Detailed Description

CMSIS STM32L1xx Device Peripheral Access Layer Header File.

Author
MCD Application Team
Version
V2.1.3
Date
04-March-2016 The file is the unique include file that the application programmer is using in the C source code, usually in main.c. This file contains:
  • Configuration section that allows to select:
    • The STM32L1xx device used in the target application
    • To use or not the peripheral’s drivers in application code(i.e. code will be based on direct access to peripheral’s registers rather than drivers API), this option is controlled by "#define USE_HAL_DRIVER"
